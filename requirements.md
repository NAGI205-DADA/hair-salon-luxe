# 要件定義書 — hair salon LUXE ランディングページ

## 1. サイト概要

| 項目 | 内容 |
|------|------|
| サイト名 | hair salon LUXE |
| サイト種別 | 1ページ構成ランディングページ（LP） |
| 目的 | 新規顧客の獲得・予約・問い合わせ受付 |
| 運営形態 | 個人経営美容室 |

---

## 2. ターゲット

- **メインターゲット：** 20代〜40代の女性
- **特徴：** スマートフォンでの閲覧が多い、SNSでサロンを探す層
- **ニーズ：** 仕上がりイメージの確認、料金の透明性、予約のしやすさ

---

## 3. ページ構成

1ページのみ（スクロール型LP）。ナビゲーションはヘッダーの固定メニューで各セクションへアンカーリンクする。

---

## 4. 各セクションの内容

### 4-1. Hero（ファーストビュー）
- キャッチコピーとサブコピー
- サロンのメインビジュアル画像
- 予約フォームへ誘導するCTAボタン

### 4-2. About（サロンについて）
- サロンのコンセプト・こだわりのテキスト
- 内装や雰囲気がわかるサブビジュアル

### 4-3. Menu / Price（メニュー・料金）
- 施術カテゴリ別のメニュー一覧（例：カット、カラー、パーマ、トリートメントなど）
- 各メニューの料金（税込表示）
- 所要時間の目安

### 4-4. Gallery（ギャラリー）
- スタイル写真を格子状に配置
- クリックで拡大表示（ライトボックス）

### 4-5. Staff（スタッフ紹介）
- スタイリストの顔写真
- 名前・役職・得意なスタイル・一言コメント

### 4-6. Access（アクセス）
- 店舗住所・最寄り駅からのアクセス
- Googleマップの埋め込み
- 営業時間・定休日

### 4-7. Booking / Contact（予約・お問い合わせ）
- 予約フォーム：名前、電話番号、希望日時、希望メニュー、備考
- 問い合わせフォーム：名前、メールアドレス、お問い合わせ内容
- 送信ボタン（フォームのバリデーションあり）

### 4-8. Footer
- サロン名・コピーライト表示
- SNSアイコンリンク（Instagram、X など）

---

## 5. 技術要件

| 項目 | 内容 |
|------|------|
| 使用言語 | HTML / CSS / JavaScript のみ |
| フレームワーク | 使用しない（ライブラリも原則禁止） |
| ファイル構成 | `index.html` / `style.css` / `script.js` |
| 画像素材 | ダミー画像を使用（本番運用時に差し替え） |
| レスポンシブ対応 | スマートフォン・タブレット・PCに対応 |
| ブレークポイント | 768px（SP/PC の切り替え目安） |
| 文字コード | UTF-8 |
| コメント記述言語 | 日本語 |
| ファイル命名規則 | ケバブケース（例: `hero-bg.jpg`） |

---

## 6. 対応範囲

- 静的HTMLファイルによるページ制作
- CSS によるスタイリング・アニメーション
- JavaScript によるフォームバリデーション・ライトボックス・スムーススクロール
- レスポンシブデザイン対応
- Googleマップの埋め込み（iframeによる静的埋め込み）

---

## 7. 対応範囲外

- フォームの送信処理・バックエンド連携（サーバーサイド実装なし）
- メール送信機能（フォーム送信時のメール通知・自動返信を含む）
- CMSや管理画面の構築
- 予約管理システムとの連携
- SEO対策（メタタグの基本設定は行うが、本格的なSEO施策は含まない）
- 多言語対応
- 本番環境へのデプロイ・サーバー契約
