## 3. ページ構成

1ページのスクロール型LPとして構成する。  
各セクションへはヘッダーのナビゲーションからアンカーリンクで移動できる。

```
ヘッダー（固定ナビゲーション）
  ↓
ファーストビュー  ................. #home
  ↓
コンセプト（3つの強み）  .......... #concept
  ↓
メニュー・料金表  ................. #menu
  ↓
スタイリスト紹介  ................. #stylist
  ↓
ご予約・お問い合わせフォーム  ..... #reservation
  ↓
フッター
```

---

## 4. 各セクションの内容

### 4-1. ヘッダー

- サロン名ロゴ（`hair salon LUXE`）を左側に表示
- ナビゲーションメニューを右側に表示
  - リンク項目：コンセプト／メニュー・料金／スタイリスト／ご予約
- ページスクロール中も画面上部に固定表示（position: fixed）
- モバイル時はハンバーガーメニューに切り替え（CSS のみで実装）

---

### 4-2. ファーストビュー

| 要素 | 内容 |
|------|------|
| キャッチコピー | 「あなただけの特別な時間を」 |
| サブテキスト | サロンの特徴を一言で補足するコピー |
| 背景 | CSS グラデーションによる雰囲気表現（実写画像なし） |
| CTA ボタン | 「ご予約はこちら」→ `#reservation` へスクロール |
| 高さ | 画面高さいっぱい（100vh） |

---

### 4-3. コンセプト（3つの強み）

3 つの強みをカード形式で横並びに表示する。

| # | 強み | 概要 |
|---|------|------|
| 01 | 完全個室のプライベート空間 | 他の客を気にせずリラックスできる個室を完備 |
| 02 | オーガニック製品を使用 | 厳選したオーガニック認定製品のみを使用 |
| 03 | 駅徒歩3分の好アクセス | 渋谷駅から徒歩 3 分の好立地 |

## 6. 対応範囲と対応範囲外

### 対応範囲（スコープ内）

- HTML・CSS による静的ランディングページの実装
- 全 7 セクション（ヘッダー〜フッター）の UI・レイアウト
- レスポンシブデザイン（PC・タブレット・モバイル）
- CSS アニメーション（ヒーローテキストのフェードイン、ホバーエフェクト等）
- CSS のみによるハンバーガーメニュー（モバイル）
- ページ内スムーズスクロール（CSS `scroll-behavior: smooth`）
- 予約フォームの UI 実装（HTML 属性による基本バリデーション含む）
- Google Fonts の利用

### 対応範囲外（スコープ外）

| 項目 | 備考 |
|------|------|
| フォームの送信処理・メール送信機能 | 外部サービス（Formrun 等）を別途利用すること |
| JavaScript を使用した動的処理 | 技術要件により JS 禁止 |
| 実写写真・本番用画像の調達 | 別途ご用意ください |
| テキスト（コピー）の作成 | 仮テキストを使用。実際の文言は別途ご提供ください |
| Google マップの埋め込み | フッターへの地図設置は対象外 |
| SEO 対策（構造化データ、OGP 等） | 基本 meta タグのみ対応 |
| CMS・管理機能 | 静的 HTML ファイルのみで構成 |
| サーバーへのアップロード・公開作業 | デプロイ作業は対象外 |
| IE 等のレガシーブラウザ対応 | モダンブラウザ最新版（Chrome・Firefox・Safari・Edge）のみ対象 |
